# Slides Folder

## Presentation Materials

This folder contains the presentation slides for the RetainAI project.

### Contents

- **Demo Slides** (10-minute presentation)
- **Technical Deep-Dive** (optional extended version)

### Recommended Slide Structure

1. **Opening** (1-2 min)
   - The turnover problem: "25% of employees leave annually"
   - Human story: The HR director who lost their best engineer

2. **Problem Statement** (2-3 min)
   - The Black Box Problem
   - The Privacy Paradox
   - The Bias Trap

3. **Solution Overview** (3-4 min)
   - Three pillars: Explainable + Privacy + Fairness
   - Product demo screenshots

4. **Demo** (3-4 min)
   - Live notebook walkthrough
   - Show prediction + explanation

5. **Impact & Close** (1-2 min)
   - Cost savings
   - Trust built-in
   - Call to action

### File Formats Supported

- PowerPoint (.pptx)
- PDF (.pdf)
- Google Slides
- Keynote (.key)

### Tips for Delivery

- Practice the demo flow beforehand
- Have backup slides ready
- Focus on the "why" not just the "how"
- End with a clear call to action
